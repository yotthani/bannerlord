    ui_ring_system_1.tpac

the sprites that are defined in:

    GUI/HeirOfNumenorSpriteData.xml

  - ring_table_clean
  - ring_table_with_tools
  - rings\ring_one

  - rings\ring_mortal_1 through rings\ring_mortal_9
