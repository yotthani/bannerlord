# Heir of Numenor - Bannerlord Overhaul Mod

## Overview
A comprehensive Lord of the Rings-themed overhaul for Mount & Blade II: Bannerlord.

## Features

### Core Systems
- **Ring System** - 20 Rings of Power with unique abilities, orbital UI, corruption mechanics
- **Custom Resources** - Culture-specific resource requirements
- **Memory System** - Troop progression and captain promotions
- **Troop Status** - Health, morale, and fatigue tracking

### Combat Enhancements
- **Battle Action Bar** - Context-sensitive commands (Shield Wall, Pike Wall, Testudo, etc.)
- **Smart Cavalry AI** - Coordinated charges, line formation, auto-reform
- **Layered Armor** - Gambeson + Chainmail + Plate protection
- **Siege Auto-Dismount** - Automatic horse handling for sieges

### Management
- **Formation Presets** - Save and load battle formations
- **Equipment Presets** - Quick equipment loadouts
- **Fief Management** - Enhanced settlement control
- **Smithing Extended** - Armor crafting and repair

### Quality of Life
- **Transfer Button Menu** - Bulk inventory actions
- **Mixed Formations** - Custom unit positioning

## Installation
1. Install dependencies: Harmony, MCM
2. Copy to `Modules/HeirOfNumenor`
3. Enable in launcher

## Requirements
- Bannerlord v1.2.0+
- Harmony
- Mod Configuration Menu (MCM)
